# Pad.IX, a custom 9 key macropad

this project is a spot where the code, schematics, 3d models, and other files related to the Hack Club Stardance "HackPad" project are located, with my iteration named "Pad.IX" since it has nine keys. 

## compentents
    - 9x cherry MX switches
    - 1x 0.92 OLED display
    - 1x Seed XIAO RP2040
    - 1x Rotary Encoder
    - 1x Custom designed and manufactured pcb
    - 1x 3d printed top housing
    - 1x 3d printed bottom housing
    - 10x M3 heatpress inserts
    - 10x m3 25mm bolts with nuts

## Software
    - Hopefully QMK but IDK
    - Fusion 360
    - KiCad 9.0
